## Synthesis for our approach

The crowd verdict is consistent: **flat tabular models plateau; the winners exploit sequence
and spatial structure.** The highest-leverage pipeline is:
1. Per-well linear prior `tvt ≈ linear(MD, Z)` (hengck23 stage.1) as a baseline + residual base.
2. Align lateral `GR` to the typewell `GR` template via DTW / correlation to read geologic
   position (hengck23 DWT thread; typewell-is-the-label-space).
3. 1D-CNN sequence model over `GR`/geometry to predict `tvt` (or the residual off the prior),
   i.e. log-inversion (hengck23 MTP thread; nvidia-kaggle teardown).
4. Ensemble the physics-y prior + DTW match + CNN. Uncertainty estimation is explicitly
   rewarded in the Working Note criteria.
